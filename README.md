# Tic Tac Toe — C (Terminal)

A compact, two-player Tic Tac Toe (Noughts & Crosses) game implemented in C for the terminal.  
Clear ASCII board rendering, robust input validation, and automatic win/draw detection make this ideal for learning C basics or playing a quick game in the console.

## Features

- Two-player local play (X vs O)
- Simple and readable implementation in plain C
- Input validation (range check, occupied-slot check)
- Automatic detection of wins and draws
- Minimal dependencies — built with any standard C compiler

## Files

- `tic_tac_toe.c` — Complete game source code

## Requirements

- A C compiler (GCC, Clang, or compatible)

## Build & Run

Compile and run from a terminal:

```bash
gcc tic_tac_toe.c -o tic_tac_toe
./tic_tac_toe
